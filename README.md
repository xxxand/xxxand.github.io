# 游戏合集

纯静态 HTML/CSS/JS 小游戏合集，部署在 GitHub Pages。

**在线访问**: [https://xxxand.github.io](https://xxxand.github.io)

## 游戏列表

| 游戏 | 说明 |
|------|------|
| 贪吃蛇 | 经典贪吃蛇，键盘/触屏控制，三档速度 |
| 2048 | 数字合并，键盘/触屏滑动，支持悔一步 |
| 五子棋 | 15×15 棋盘，双人对战 / 人机对战 (Alpha-Beta AI)，禁手规则可选，支持悔棋 |
| 扫雷 | 9×9 10 雷，点击揭开，长按/右键标旗 |

## 本地打开

直接双击 `index.html` 即可，无需安装任何工具。

## 目录结构

```
index.html          # 首页
snake.html          # 贪吃蛇
2048.html           # 2048
gomoku.html         # 五子棋
minesweeper.html    # 扫雷
css/
  common.css        # 公共基础样式
  index.css         # 首页样式
  snake.css         # 贪吃蛇样式
  2048.css          # 2048 样式
  gomoku.css        # 五子棋样式
  minesweeper.css   # 扫雷样式
js/
  snake.js          # 贪吃蛇逻辑
  2048.js           # 2048 逻辑
  gomoku.js         # 五子棋逻辑
  minesweeper.js    # 扫雷逻辑
```
